# Agenciador de Cargas Rodoviárias

Sistema inicial para agenciamento de cargas rodoviárias, busca de motoristas, controle de veículos, negociação de fretes e acompanhamento operacional.

## Stack

- Next.js
- React
- TypeScript
- Vercel

## Módulos da V1

- Dashboard operacional
- Cadastro/listagem de cargas
- Cadastro/listagem de motoristas
- Controle de veículos
- Negociações de frete
- API de health check
